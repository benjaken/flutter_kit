# Flutter 组件库

Flutter Kit 是仿[Vant](https://youzan.github.io/vant/#/zh-CN/intro)组件库的flutter widget库

## 项目介绍

### 如何使用

```
dependencies:
  flutter_vant_kit: ^0.2.1
```

### 运行Demo

```
cd example/
flutter run
```

## API

详见 [文档](https://benjaken.gitbook.io/flutterzu-jian-k/) 或 [doc文件夹](./doc)
