## Panel 面板

### 使用范例

```
import 'package:flutter_vant_kit/main.dart';

Panel(
  title: "标题",
  desc: "描述信息",
  ...
)
```

更多例子请参考[Demo](https://github.com/benjaken/flutter_vant_kit/blob/master/example/lib/routes/demoPanel.dart)

### API

| 参数 | 说明 | 类型 | 默认值 |
| ------------ | ------------ | ------------ | ------------ |
| title | 标题 | `String` | - |
| desc | 描述 | `String` | - |
| status | 状态 | `String` | - |
| body | 自定义内容 | `Widget` | - |
| footer | 自定义 footer | `Widget` | - |
